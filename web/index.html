Function Points

Function points are used to gain an estimate of time by determining the estimated complexity and number of modules within a project as well as the influence of various technical factors.

Basic COCOMO

Basic COCOMO is the quickest method for gaining an estimate of time for a project. It references the overall size and complexity of the project.

Intermediate COCOMO

A more refined version of basic COCOMO. In addition to a referenced size, it uses various cost drivers to determine the estimated amount of time.
